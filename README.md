# Physics in Action

**AP Physics C concepts applied to real-world engineering projects.**

This is my public portfolio showcasing hands-on projects that connect classroom physics with practical engineering and robotics.

### Current Main Project
**Wearable Assistive Hand Gripper**  
A servo-driven, tendon-based wearable robotic gripper designed to assist people with weak grip strength.

- **CAD**: Onshape (full educational license until graduation)  
- **Fabrication**: School 3D printer  
- **Electronics**: Arduino + SG90/MG90S servo + fishing line tendons  
- **Physics Applied**: Torque, levers, mechanical advantage, rotational dynamics (AP Physics C)  
- **Status**: Development begins May 15, 2026 (after AP exams)  
- **Goal**: Fully functional prototype by December 2026 for Purdue University FYE Regular Decision application
